1. Fundação do Projeto: Crie a estrutura de pastas, as interfaces (IAgent, ITool, etc.) e o serviço de configuração.
2. Construir o Núcleo da Orquestração: Implemente o AgentOrchestratorFacade e o StateManager (usando uma versão em memória). Crie um script simples (run.ts ou um teste) que chame a fachada diretamente para provar que o ciclo de vida (iniciar, executar, completar) funciona.
3. Implementar a Inteligência (Agentes): Crie os Adapters para os LLMs e os agentes reais (PlannerAgent, ResearcherAgent). Use seu script de teste para acionar um fluxo completo, desde o planejamento até a execução de uma tarefa, validando a lógica no console.
4. Adicionar o Sistema de Ferramentas: Implemente a infraestrutura para as ferramentas e crie uma ferramenta funcional para ser usada pelo agente de execução.
5. Expor o Núcleo via API: Com toda a lógica de negócio testada e funcionando, crie a camada final: o servidor HTTP (Express) e o SSEManager. O controlador da API simplesmente receberá as chamadas e delegará para a fachada, que já está pronta e validada.
